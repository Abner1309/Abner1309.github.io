Lista de Contatos

Linguagem: Java

Bibliotecas-Chave: JavaFX + SceneBuilder + JDBC

Status: Concluído

Equipe: Solo

Tipo de Projeto: Pessoal

Link do Projeto: GitHub

Descrição

Este projeto consiste na implementação de um aplicativo desktop simples e intuitivo para gerenciamento de contatos.

Funcionalidades

  • Registro de contatos (Nome, Telefone e E-mail)
  • Edição de contatos existentes.
  • Busca ativa de contatos.
  • Exclusão de contatos.
  • Customização da interface.

Tecnologias Usadas

  • Java 21+ (Linguagem de Programação)
  • JavaFX 21+ (Framework de Interfaces)
  • Maven (Gerenciador de Dependências)
  • Scene Builder (Construtor FXML)
  • CSS (Estilização)
  • SQLite (Banco de Dados)
