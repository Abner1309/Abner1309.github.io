Processamento de Linguagem Natural

Linguagem: Python

Bibliotecas-Chave: LangChain e Google Gemini API

Status: Concluído

Equipe: 2 Desenvolvedores

Tipo de Projeto: Acadêmico

Link do Projeto: Repositório Privado. Caso tenha interesse, entre em contato comigo.

Descrição

Esse projeto consiste na implementação de um notebook no Google Colab que faça uso do Framework LangChain e de um LLM aplicando as duas técnicas abaixo de PLN na página de avaliações dos usuários do site IMDb.

  • Tradução de Textos.
  • Análise de Sentimentos.

Funcionalidades

  • Web Scrapping na Página Web do IMDb.
  • Tradução do texto (Inglês -> Português) extraído através do LangChain + Google Gemini.
  • Análise de sentimentos do texto traduzido para o português.
  • Análise gráfica dos resultados.
