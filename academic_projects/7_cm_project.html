Compilador de Uma Calculadora

Linguagem: Java

Bibliotecas-Chave: Java.Util.Scanner

Status: Concluído

Equipe: 1 Desenvolvedor

Tipo de Projeto: Acadêmico

Link do Projeto: GitHub

Descrição

Esse projeto consiste na implementação de um compilador para uma calculadora simples. Dessa forma, para alcançar esse objetivo foi necessário a implementação de um analisador léxico para a criação de tokens a partir da string de entrada e de um analisador sintático descendente recursivo para a verificar possíveis erros sintáticos dos tokens.

Funcionalidades

  • Análise Léxica.
  • Análise Sintática.
  • Análise Semântica Parcial.
