Representação de Números Com Muitos Algarismos

Linguagem: C

Bibliotecas-Chave: StandardIO e StandardLib

Status: Concluído

Equipe: 3 Desenvolvedores

Tipo de Projeto: Acadêmico

Link do Projeto: GitHub

Descrição

Esse projeto consiste em implementar um algoritmo que seja capaz de representar números com muitos algarismos. A importância desse projeto consiste no fato de que muitas áreas da computação, como é o caso da criptografia fazem uso de números gigantescos (10³⁰⁰) ou mais. Nesse sentido, as variáveis float/double não são capazes de armazenar números dessa magnitude.

Funcionalidades

  • Representação de Um Big-Number.
  • Operação de Soma Com Big-Number.
  • Operação de Subtração Com Big-Number.
  • Operação de Multiplicação Com Big-Number.
