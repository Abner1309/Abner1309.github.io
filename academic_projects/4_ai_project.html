Agente Inteligente no Super Mario World

Linguagem: Python

Bibliotecas-Chave: NEAT e Gym Retrô

Status: Concluído

Equipe: 3 Desenvolvedores

Tipo de Projeto: Acadêmico

Link do Projeto: GitHub

Descrição

Esse projeto consiste em implementar um agente inteligente capaz de jogar a fase YoshiIsland2 do jogo Super Mario World utilizando o algoritmo genético NEAT (NeuroEvolution of Augmenting Topologies) que é focado na criação de redes neurais evolutivas.

Funcionalidades

  • Criação de Rede Neural Evolutiva.
  • Treinamento de Um Agente Inteligente.
  • Conclusão do Cenário YoshiIsland2.
