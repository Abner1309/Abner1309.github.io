Status: Concluído

Link do Projeto: Google Colab

Descrição

Considere a base de dados disponível no endereço:
https://drive.google.com/u/1/uc?id=1TPHQyck1Rdi8lwzLOkBaI0DvkqmJJdG-&export=download
Essa base de dados contem informações sobre a evolução, ao longo de diferentes anos, de diversos indicadores para diferentes países. Essa base foi utilizada no documentário Don’t Panic – The Facts About Population, que foi transmitido pela rede britânica BBC.
Essa base tem as seguintes colunas:

  • País (nome do país).
  • Ano (ano em que o índice foi coletado).
  • População (número de habitantes daquele país naquele ano).
  • Continente (continente em que o país está localizado).
  • Expectativa de vida.
  • PIB per capita.
  • Produto Interno Bruto.

Com base nesses dados, será realizada uma análise com base nas bibliotecas Pandas e MatPlotLib do Python.

Funcionalidades

  • Análise de Correlação.
  • Estátistica Descritiva.
  • Análise Gráfica.
